Microservices VM
================

Deploys a minimal [Microservices](http://en.wikipedia.org/wiki/Microservices) platform to a UNIX VM to enable services to be deployed using [pio](https://github.com/pinf-io/io.pinf.pio).

Compatibility
-------------

### Operating Systems

  * OSX (client)
  * Ubuntu (server)

### Service Languages

  * PHP
  * NodeJS
